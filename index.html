AVIÓN EN REALIDAD AUMENTADA CON DATOS IOT

Unity/ ARCore/ Arkit

Fecha: 24/04/2018

ESTE PROYECTO LO PUEDES VER EN EL SIGUIENTE ENLACE: Facebook

Este proyecto fue el resultado de 3 meses de arduo trabajo. Durante la creación de este proyecto se presentaron diversas dificultades en el desarrollo de esta experiencia.

Primeramente, el avión, se alimentaba con datos en tiempo real de un giroscopio, ya que el avión se emparejaba con un avión físico a escala hecho con manufactura aditiva. El hacer todo el emparejamiento y la sincronización en tiempo real fue un gran reto.

Fue un gran orgullo haber podido hacer este proyecto ya que la tecnología de Realidad Aumentada no tiene la importancia que debería en México.

AUTOMÓVIL EN REALIDAD AUMENTADA CON DATOS IOT

Unity / ARKit/ C#/ nodejs

Fecha: 09/10/2019

ESTE PROYECTO LO PUEDES VER EN EL SIGUIENTE ENLACE: Facebook

Este proyecto fue enfocado al área automotriz en el evento de strong Hannover Mess en la ciudad de León, Guanajuato.

El proyecto que se nos encargó fue el realizar un automóvil de tecnología en Realidad Aumentada, donde a través de un servicio reconociera los datos que mandaba un auto a escala; el cual contenía sensores y actuadores.

Los datos que se mandaban en esos dispositivos se tenían que plasmar en Realidad Aumentada a través de un socket hecho en node.js, donde se logró hacer lo más cercano a un Digital Twin.
